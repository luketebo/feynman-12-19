# 费曼宠物 (Feynman Pet AI)

费曼宠物是一个基于 **费曼学习法** 的 AI 电子宠物系统。用户通过向宠物解释复杂的概念来教导它，宠物会根据用户的解释质量给予反馈、获得经验并记住知识。

## 核心功能

- **费曼学习逻辑**：AI 扮演一个好奇但理解力有限的小学生，引导用户深入浅出地解释知识。
- **多宠物 & 多会话**：用户可以领养多个宠物，每个宠物拥有独立的等级和跨会话的“知识记忆”。
- **知识提取系统**：AI 会自动从对话中提取核心知识点并持久化存储，宠物在后续对话中会提及已学到的内容。
- **游戏化成长**：教导成功可获得经验值（XP）和金币（Coins）。
- **皮肤商城**：使用赚取的金币为宠物购买个性化皮肤。
- **流式交互**：基于 SSE (Server-Sent Events) 的实时打字机对话效果。

## 技术栈

- **后端**: Flask (Python)
- **数据库**: MySQL (远程连接)
- **AI 能力**: OpenAI API (GPT 系列模型)
- **前端**: 原生 HTML/CSS/JS (响应式设计)
- **包管理**: uv

## 快速开始

1. **安装依赖**:
   ```bash
   uv sync
   ```

2. **配置环境**:
   在 `.env` 文件中配置以下变量：
   - `DATABASE_URL`: MySQL 连接字符串
   - `OPENAI_API_KEY`: 你的 API 密钥
   - `OPENAI_BASE_URL`: API 代理地址（可选）

3. **运行项目**:
   ```bash
   python run.py
   ```

## 数据库结构

项目采用模块化设计，包含 `User`, `Pet`, `Session`, `Message`, `Skin` 等核心模型。详见 `API_DOCS.md`。
